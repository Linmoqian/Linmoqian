## 关于我

二十岁，站在两个世界的交汇处——  
一边是显微镜下细胞分裂的秩序与奇迹，一边是屏幕前代码流淌的逻辑与韵律。

我对两个问题始终着迷：

- **生命如何编码自身**
- **智能如何作用于现实**

因此，我的兴趣逐渐汇聚到三条主线：

- **AI for Biology**：用深度学习理解生命信息与生物序列
- **Embodied Intelligence**：让智能体从“推理”走向“行动”
- **Modular Systems**：构建可组合、可扩展、可复用的工程系统

我希望做的不只是“能跑起来”的代码，  
而是**能够承载思想、映射现实、并持续演化的系统**。

## 精选项目

### 1. [DNA_CNN_predict](https://github.com/Linmoqian/DNA_CNN_predict)
**CNN 预测 DNA 表达**

基于卷积神经网络，从 DNA 序列中预测表达模式，探索序列结构与生物功能之间的映射关系。

**Tech Stack:** `Python` `PyTorch` `CNN` `Bioinformatics`

**项目亮点：**
- 面向 DNA 序列建模任务
- 包含数据预处理、模型训练与评估流程
- 关注生物序列中的局部模式识别能力

---

### 2. [claw_arm](https://github.com/Linmoqian/claw_arm)
**Agent 控制机械臂**

尝试将智能体逻辑接入机械臂控制流程，让决策系统与物理执行系统形成闭环。

**Tech Stack:** `Python` `Agent` `Robotics` `Control`

**项目亮点：**
- 面向具身智能与智能控制场景
- 关注“感知—决策—执行”的系统联动
- 探索 Agent 在现实任务中的落地可能性

---

### 3. [block-builder](https://github.com/Linmoqian/block-builder)
**积木化深度学习系统**

尝试将深度学习流程进行模块化拆解与组合，降低实验成本，提升系统复用性与扩展性。

**Tech Stack:** `Python` `Deep Learning` `System Design`

**项目亮点：**
- 强调模型训练流程的模块化设计
- 面向实验管理与快速迭代
- 关注“构建方式”而不仅是“单次结果”

## 我在思考什么

ATCG 写着生命最古老的程序，  
每一对碱基，都是一次跨越亿万年的信息编码。

0 与 1 承载着新时代的脉搏，  
每一个循环、每一次梯度更新，都在逼近某种“人工理解”的边界。

当显微镜的目镜与显示屏的光芒重叠时，我看到的是：

- 一条流淌了四十亿年的河流：从单细胞到意识
- 一条只走了半个世纪的河流：从计算到创造

真正的奇迹，不发生在单一学科的中心，  
而发生在那些**边界模糊、碰撞激荡、尚未被命名的交界地带**。

### 🌟 Philosophy

In my world, two rivers converge:

**🧬 DNA is Code** — Four bases writing life's most ancient program, each pair a carefully designed function call

**🤖 Code is Life** — Between 0 and 1 flows the pulse of a new era, every loop a rhythm of breath

**✍️ Pen is Heart's Extension** — Writing from the heart, driving the pen with intention. Text is not a tool, but the soul's breath, the projection of thought onto paper

**⚙️ Engineering is Thought Incarnate** — Engineering is the projection of theory into reality. With code and steel, we forge abstract ideas into tangible truth

When the microscope's eyepiece meets the display's glow, I see—

One river flowing for four billion years, from single cell to human, from chaos to consciousness;
One river flowing for half a century, yet reaching the stars, from computation to creation.

The true miracle happens in that boundary-blurring, collision-stirring sea. There, biology meets code, reason meets creation, known meets unknown.

## ⚙️ Engineering Practice

> **"Code is written for humans to read, only incidentally for machines to execute."**

I believe elegant code is the best documentation, and standardized processes are the cornerstone of team collaboration.

### 📐 Code Standards

| Language | Standard | Tools |

| React | React Official | `react-devtools` |

### 🧪 Testing Philosophy

```
Testing is not a burden, but a lighthouse of security in late nights.

├── Unit Testing  —— pytest/jest, coverage > 80%
├── Integration Test  —— API end-to-end verification
├── Performance Test  —— Stress testing, benchmarking
└── TDD Practice  —— Red, then Green, then Refactor
```

### 🌳 Version Control

```
Git Flow Workflow
├── main      —— Production
├── develop   —— Development branch
├── feature/* —— Feature development
└── hotfix/*  —— Emergency fixes

Commit Convention: Conventional Commits (Chinese descriptions)
feat: New feature
fix: Bug fix
docs: Documentation
refactor: Refactoring
test: Testing
chore: Build/tools
```

👨‍💻 关于我 (About Me)

我是 陈林峰，一名就读于 华南农业大学 (SCAU) 的生物科学专业本科生。我不只是在实验室里跑 PCR，我还在服务器上跑模型，在电路板上跑电流。我致力于探索 AI + Biotech + Robotics 的交叉领域。

🔭 我目前正在研究 GEM 基因组规模代谢模型 与 机器人 SLAM 建图。

🌱 我正在深入学习 大模型微调 (LLM Fine-tuning) 和 ROS2。

👯 我正在寻求 生物信息学 或 具身智能 相关的开源合作机会。

💬 关于 Python, STM32, 深度学习 的问题，欢迎随时问我。

⚡ Fun fact: 我可以用 Python 分析 DNA 序列，也可以用它控制机械臂抓取试管。
